#什麼是Warp？

一個AI的terminal，可以讓你在terminal中使用AI，並且可以讓你的AI在terminal中使用你的電腦。
還有許多功能可以讓開發變得更容易，協作上更方便。

背後技術：

語言：使用Rust為原生語言
AI：使用OpenAI的 API

#方案

免費版(Free)

  1. 一天AI可用20個request
  2. 可分享
  3. 有離線功能
  4. 5人以內共享指令的team功能(目前為免費使用，預計在2023年十月收費)

價錢: 0元


付費版(Team)

  1. 一天AI可用100個request
  2. 即時分享(未釋出)

價錢: 12元(美金)/人/月

#限制與潛在問題(相對應的緩解辦法)

  1. 不支援windows：
    目前只支援MAC系統，但在未來會延伸到Linux以及Windows等。

  2. 不完全支援中文(輸入注音顯示不出來)：
    中文可以顯示，但是輸入注音會顯示不出來。

  3. AI有時候也無法解決遇到問題：
    測試了幾個實際會遇到的情境，AI這邊沒有辦法得到正確精確的答案，
    要觀察未來是否可能會更精確，更容易解決問題。

  4. 協作雖然可以分享指令，但有些指令可能需要各自設定調整：
    雖然有變數可以調整，但實際情境可能會因為電腦不同環境不同，需要不同的調整，增加減少指令，這樣會讓協作上面沒辦法達到原本使用這個工具的最大效益。

  5. 需要登入才可以使用

#想法&結論

  • 操作介面直覺好上手，遇到複雜的指令且需要重複使用，他可以幫助減少錯誤跟記憶指令的時間。
  • AI的部分可以幫助快速理解指令以及基本的除錯，但也有情境是沒辦法直接透過他解決的。
  • 團隊上目前看起來最有幫助的用法會是：
    1. 以渣打來看，包版還有後續上架的程序，可以統整，讓整個過程更順暢，減少人為錯誤。
    2. 未來專案也可以使用它來讓整個程序更有效率順暢，新進專案的人也可以更快上手。
  • 團隊分享的功能的確是有幫助，未來可以觀察付費版本正式釋出時，會不會有更多團隊會用到的功能。
